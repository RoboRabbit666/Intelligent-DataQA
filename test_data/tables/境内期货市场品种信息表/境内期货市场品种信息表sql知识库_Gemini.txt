问题：查询2025年7月2日郑州商品交易所的所有期货品种。
SELECT CLAS_NAME
FROM dataqa.fut_mrkt_clas_info_tab
WHERE BIZ_DT = '2025-07-02' AND EXCH_CODE = 'CZCE' AND fut_opt_flag = 'F';

问题：查询2025年7月上海期货交易所成交量最大的前5个期货品种。
SELECT CLAS_NAME, SUM(trd_qty) as total_trd_qty
FROM dataqa.fut_mrkt_clas_info_tab
WHERE EXCH_CODE = 'SHFE' AND BIZ_DT >= '2025-07-01' AND BIZ_DT <= '2025-07-31' AND fut_opt_flag = 'F'
GROUP BY CLAS_NAME
ORDER BY total_trd_qty DESC
LIMIT 5;

问题：统计每个交易所的品种数量。
SELECT EXCH_NAME, COUNT(DISTINCT CLAS_CODE) as number_of_products
FROM dataqa.fut_mrkt_clas_info_tab
GROUP BY EXCH_NAME;

问题：找出所有交易所的苹果(AP)期货在2025年7月2日的总成交额。
SELECT SUM(trd_amt)
FROM dataqa.fut_mrkt_clas_info_tab
WHERE BIZ_DT = '2025-07-02' AND CLAS_CODE = 'AP' AND fut_opt_flag = 'F';

问题：2025年第一季度，哪个品种的月均成交量最高？
SELECT CLAS_NAME, AVG(trd_qty) as avg_monthly_volume
FROM dataqa.fut_mrkt_clas_info_tab
WHERE BIZ_DT BETWEEN '2025-01-01' AND '2025-03-31'
GROUP BY CLAS_NAME
ORDER BY avg_monthly_volume DESC
LIMIT 1;

问题：2025年7月2日，哪些品种的期货和期权都有交易？

问题：查询2025年7月2日，所有品种的主力合约成交量占总成交量的比例。
SELECT CLAS_NAME, SUM(main_comd_trd_qty) * 100.0 / SUM(trd_qty) as main_contract_volume_ratio
FROM dataqa.fut_mrkt_clas_info_tab
WHERE BIZ_DT = '2025-07-02' AND trd_qty > 0
GROUP BY CLAS_NAME;

问题：统计2025年7月2日，每个交易所的期货和期权合约的数量分别是多少？
